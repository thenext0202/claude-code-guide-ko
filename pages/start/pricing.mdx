# 03 · 요금제와 비용

> "월 얼마 내야 써요? 깐 다음에 결제해야 한다는데."

가장 헷갈리는 부분이라 한 번에 정리할게요. **2026년 4월 기준**입니다 (가격은 변동 가능, [공식 페이지](https://www.anthropic.com/pricing)에서 최신 확인).

## 한 눈에 보는 비교

| 요금제 | 월 가격 | 한도 | 누구를 위한? |
|---|---|---|---|
| **Free** | 무료 | 매우 제한적 | 맛보기, 가벼운 채팅용 |
| **Pro** | $20/월 | Free보다 5배+ | **개인 사용자 추천** |
| **Max 5x** | $100/월 | Pro의 5배 | 매일 쓰는 헤비 유저 |
| **Max 20x** | $200/월 | Pro의 20배 | 풀타임 코딩 동반자 |
| **Team** | $25/인/월 | 팀 단위 협업 | 회사·스튜디오 |
| **API (종량제)** | 사용한 만큼 | 무제한 | 개발자가 자기 앱에 붙일 때 |

> 💡 Claude Code는 **Pro 이상부터** 본격적으로 쓸 수 있습니다. Free에선 거의 못 쓴다고 보면 돼요.

## 어떤 걸 골라야 해요?

### 시나리오별 추천

**🌱 처음이라 일단 써보고 싶음 → Pro ($20/월)**
- 거의 모든 사람의 시작점
- 월 한도가 넉넉해서 하루 1~2시간씩 써도 충분
- 1년 결제 시 할인

**💼 매일 업무에서 4시간 이상 → Max 5x ($100/월)**
- 회의록 정리, 코드 리팩토링, 문서 작성을 매일 하는 직장인/개발자
- 한도 걸려서 답답한 일이 거의 없어짐

**🚀 풀타임 코딩 파트너로 쓸 거 → Max 20x ($200/월)**
- 1인 개발자, 사이드 프로젝트 빌더
- 하루 8시간 곁에 두고 쓸 사람
- "한도 신경 안 쓰고 싶다" 면 답

**🏢 회사에서 동료들과 → Team ($25/인/월, 최소 5인)**
- 관리 콘솔, 사용량 분석, 통합 결제
- IT 팀 입장에서 좋음

**🛠 API에 직접 호출할 거 → API 종량제**
- 자기 앱에 Claude를 임베드할 때
- 토큰당 과금 ([API 가격표](https://www.anthropic.com/pricing#api))
- Claude Code 자체는 Pro/Max에 포함이라 별도 API 키 안 필요

## "토큰" 이 뭔데 한도가 걸려요?

> 토큰 = AI가 글자를 세는 단위 (대충 단어 1개 = 1.3토큰)

Claude Code는 대화하면서 **토큰을 소모**합니다. 요금제마다 한 시간/하루/주마다 쓸 수 있는 토큰 양이 정해져 있어요.

### 토큰을 많이 쓰는 일
- 큰 파일 통째로 읽기 (PDF 100페이지 → 토큰 잔뜩)
- 긴 대화 이어가기 (이전 내용도 매번 다시 보냄)
- 결과를 길게 받기

### 토큰을 적게 쓰는 일
- 짧은 질문 / 짧은 답변
- 새 세션 시작 (이전 맥락 안 가져옴)
- `.gitignore`로 봐야할 파일만 지정

자세한 절약 패턴은 [20 비용 절약 팁](/advanced/cost)에서.

## 결제 방법

1. [claude.ai](https://claude.ai) 가입 후 로그인
2. **Settings → Plan & billing**
3. Pro/Max 중 선택해서 카드 등록
4. 가입 직후부터 Claude Code에서 사용 가능

> 🇰🇷 **한국 카드도 됩니다.** Visa/Master/Amex 모두 OK. 환율은 결제일 기준 적용.

## 자주 듣는 질문

<details>
<summary><b>Q. 한 달만 써보고 해지 가능?</b></summary>

가능합니다. Pro/Max 모두 월 단위 해지 가능해요. 단, 다음 결제일 전에 해지해야 다음 달 청구가 안 됩니다.
</details>

<details>
<summary><b>Q. 하루 한도 다 쓰면 어떻게 돼요?</b></summary>

해당 시간 동안(보통 5시간 단위) 잠시 못 씁니다. 다음 리셋 타임까지 기다리거나, 한 단계 위 요금제로 업그레이드.
</details>

<details>
<summary><b>Q. Pro에서 Max로 바로 올릴 수 있어요?</b></summary>

네, 즉시 가능합니다. 결제는 일할 계산.
</details>

<details>
<summary><b>Q. Cowork이라는 게 따로 있다던데?</b></summary>

**Cowork(또는 Anthropic Cowork)** 은 Anthropic이 별도로 운영하는 팀 협업 제품입니다. Claude Code와는 다른 제품이라 헷갈리지 마세요. 이 가이드는 **Claude Code (CLI)** 만 다룹니다.
</details>

<details>
<summary><b>Q. 결제 안 해도 무료로 써볼 수 있는 방법?</b></summary>

신규 가입 시 일정량의 무료 크레딧이 주어질 수 있어요. 또 학생/오픈소스/스타트업 프로그램이 있는 시즌도 있으니 [공식 페이지](https://www.anthropic.com/pricing)를 확인.
</details>
