# 01 · Claude Code란?

## 한 문장 요약

> **Claude Code는 내 컴퓨터에서 직접 일하는 Anthropic의 AI 비서입니다.**

비유하자면 이렇게 생각해보세요.

| AI 도구 | 비유 |
|---|---|
| ChatGPT (웹) | 카페에서 만나서 조언만 듣는 친구 |
| Claude.ai (웹) | 카페에서 만나는데 서류 가방까지 들고 와서 같이 보는 친구 |
| **Claude Code** | **우리 집에 와서 같이 책상 앞에 앉아 내 파일을 직접 만지며 일해주는 친구** |

웹 챗봇은 "이렇게 해보세요" 하고 말로만 알려주죠. Claude Code는 **직접 파일을 열어서 고쳐줍니다.**

## 어떻게 동작하나요?

### 1. 터미널이라는 창에서 실행돼요

여러분 컴퓨터에 있는 **터미널(Terminal)** 앱을 열고, `claude` 라고 치면 시작됩니다.

```bash
claude
```

그러면 이런 화면이 나와요:

```
╭──────────────────────────╮
│  ✻ Welcome to Claude Code │
╰──────────────────────────╯

> _
```

여기서부터 **한국어로 말 걸면** 됩니다.

```
> 이 폴더에 뭐가 있는지 알려줘
```

### 2. 폴더(=프로젝트) 단위로 일해요

Claude Code는 **현재 위치한 폴더**를 작업장으로 삼습니다.

- `/Users/내이름/Desktop/마케팅자료` 에서 실행 → 마케팅자료 폴더 안의 파일만 다룸
- `/Users/내이름/Desktop/내사이트` 에서 실행 → 사이트 코드만 다룸

폴더가 바뀌면 작업 컨텍스트도 바뀐다고 보면 돼요. 이게 굉장히 중요한 개념인데, [07 핵심 개념](/core/concepts)에서 자세히.

### 3. 도구(Tools)를 쓸 줄 알아요

Claude Code는 단순히 글을 쓰는 게 아니라:

- 📖 파일을 **읽을 수 있고**
- ✏️ 파일을 **편집할 수 있고**
- 🖥 명령어를 **실행할 수 있고**
- 🌐 웹사이트도 **열어볼 수 있고**
- 🔗 외부 서비스(Slack, GitHub, DB 등)와도 **연결됩니다** (MCP 기능)

이걸 **Tools(도구)** 라고 부릅니다. 사람이 손으로 클릭하던 일을 AI가 대신 하는 셈이에요.

## ChatGPT와 진짜로 뭐가 다른가요?

| 상황 | ChatGPT | Claude Code |
|---|---|---|
| "엑셀 정리해줘" | "이런 식으로 해보세요" 답변만 | 실제로 파일을 열어서 정리하고 저장 |
| "이 코드 버그 고쳐줘" | 수정된 코드를 답으로 보여줌 → 내가 복붙 | 직접 파일을 수정하고 결과까지 알려줌 |
| "이 폴더 100개 파일 정리" | 못 함 | 가능. 한 번에 처리 |
| "회의록 요약해서 새 파일로" | 요약은 해줌. 저장은 못 함 | 요약 + 저장까지 자동 |
| 비밀번호/민감파일 | 업로드해야 보임 | 내 컴퓨터에서만 처리. 외부 안 나감 |

## 누구를 위한 도구인가요?

### ✅ 이런 분께 잘 맞아요

- 반복 작업을 자동화하고 싶은 사람
- 1인 개발자, 프리랜서, 기획자
- "엑셀/문서 정리하는 시간이 아깝다" 고 느끼는 직장인
- AI에게 진짜 작업을 시켜보고 싶은 사람

### ❌ 이런 경우엔 굳이 안 써도 돼요

- 단순한 1회성 질문만 할 거라면 → ChatGPT/Claude.ai 웹으로 충분
- 컴퓨터 켜는 것 자체가 어렵다면 → 터미널 적응이 좀 필요해요

## 자주 듣는 오해

<details>
<summary><b>Q. "Code" 라는 이름 때문에 개발자만 쓰는 도구 아닌가요?</b></summary>

이름은 그렇지만, 실제로는 **파일 작업이 필요한 모든 사람**에게 유용해요. 비개발자가 엑셀, PDF, 워드, 폴더 정리에 더 자주 씁니다.
</details>

<details>
<summary><b>Q. 인터넷 없으면 안 되나요?</b></summary>

네, AI 모델은 Anthropic 서버에 있어서 **인터넷 연결이 필수**입니다. 단, 작업 대상 파일은 여러분 컴퓨터에 그대로 있어요.
</details>

<details>
<summary><b>Q. 한국어 잘 알아듣나요?</b></summary>

매우 잘 합니다. Claude는 한국어 처리에 강한 모델이라 한국어로 부탁하셔도 영어로 부탁할 때와 거의 같은 품질이 나와요.
</details>
