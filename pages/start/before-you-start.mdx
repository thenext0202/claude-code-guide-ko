## 1. Claude Code는 "내 컴퓨터의 AI 동료" 입니다

가장 중요한 한 줄로 시작하겠습니다.

> Claude Code는 내 컴퓨터의 폴더 안에 들어와서, 내 파일을 읽고·고치고·정리해주는 AI입니다.

ChatGPT처럼 웹 브라우저에서 채팅하는 도구가 아니라, **내 노트북의 폴더에서 직접 일하는 동료**라고 생각하면 됩니다.

| 도구 | 어디서 일하나요? | 누구를 위한 것? |
|---|---|---|
| ChatGPT | 웹 브라우저 안 (대화창만) | 일반 질문, 글쓰기 |
| Claude.ai | 웹 브라우저 안 (파일 업로드 가능) | 채팅 + 가벼운 분석 |
| **Claude Code** | **내 컴퓨터의 터미널** | **내 폴더에 직접 들어와 작업** |

### 그래서 무엇이 다른가요

- 내 파일을 직접 읽고 수정합니다 — "이 폴더의 엑셀 파일들을 정리해줘"가 가능합니다.
- 여러 단계를 연달아 처리합니다 — 파일 읽기 → 분석 → 보고서 작성을 한 번에.
- 터미널이라는 도구를 사용합니다 — 검은 화면에 명령을 입력하는 방식. 처음엔 어색해도 5분이면 적응됩니다.

<Callout type="note" title="터미널이 무엇인가요">
컴퓨터에 마우스가 아닌 글자로 말을 거는 입력창입니다. Windows라면 [Windows 환경 풀가이드](/extras/windows-setup)에서 가장 안 막히는 셋업 경로를 정리해두었습니다.
</Callout>

## 2. 코딩을 몰라도 됩니다

이름에 "Code"가 들어가서 개발자 전용처럼 보이지만, 실제로는 **비개발자가 더 자주 쓰는 패턴**도 적지 않습니다.

비개발자도 다음과 같은 작업을 부탁할 수 있습니다.

- "이 폴더의 회의록 30개를 한 페이지로 요약해줘."
- "엑셀 시트들에서 매출 합계만 뽑아 새 파일로 저장해줘."
- "이 블로그 글들을 영어로 번역해서 새 폴더에 정리해줘."
- "다운로드 폴더를 종류별로 정리해줘. 영수증은 receipts/, 이미지는 images/."

전부 **한국어로 부탁**하면 됩니다. 영어도, 코딩도 필요하지 않습니다.

## 시작하기 전 체크리스트

| 필요한 것 | 안 되어 있다면 |
|---|---|
| 인터넷 연결 | (생략) |

| 노트북 (Mac/Windows 모두 가능) | — |

## 자주 듣는 걱정

<details>
<summary><b>내 파일을 AI가 다 읽으면 정보가 새지 않나요?</b></summary>

Anthropic은 기본적으로 **사용자 데이터로 모델을 학습시키지 않습니다.** 또한 권한 시스템(Permissions)으로 "이 파일/폴더만 봐도 됨"처럼 범위를 제한할 수 있습니다. 자세한 내용은 [07 핵심 개념](/core/concepts)에서 다룹니다.
</details>

<details>
<summary><b>코딩을 1도 모르는데 따라올 수 있나요?</b></summary>

이 가이드는 **터미널을 처음 열어보는 분 기준**으로 작성됐습니다. "검은 화면에 무엇을 입력해야 하나요?"부터 손잡고 진행합니다.
</details>

<details>
<summary><b>무료로 써볼 수 있나요?</b></summary>

Claude Code 자체는 Anthropic 요금제에 포함된 도구라 **유료 구독이 있어야** 본격적으로 쓸 수 있습니다. 다만 신규 가입 시 일정량의 무료 크레딧이 주어지는 시즌이 있으니 [공식 사이트](https://claude.ai)에서 확인하세요.
</details>

<Callout type="warning" title="Windows 사용자라면 잠깐">
설치 전에 [Windows 환경 풀가이드](/extras/windows-setup)를 먼저 읽으시면 막히는 시간을 크게 줄일 수 있습니다. WSL · Git Bash · PowerShell 중 어떤 환경에서 시작할지가 가장 먼저 정할 결정입니다.
</Callout>
