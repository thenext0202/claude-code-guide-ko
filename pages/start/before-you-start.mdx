## 1️⃣ Claude Code = 내 컴퓨터의 AI 동료

가장 중요한 한 줄입니다.

> **Claude Code는 내 컴퓨터의 폴더 안에 들어와서, 내 파일을 읽고·고치고·정리해주는 AI입니다.**

ChatGPT처럼 웹브라우저에서 채팅하는 게 아니라, 내 노트북 폴더에서 직접 일하는 친구라고 생각하면 돼요.

| 도구 | 어디서 일하나요? | 누구를 위한 것? |
|---|---|---|
| ChatGPT | 웹브라우저 안 (대화창만) | 일반 질문, 글쓰기 |
| Claude.ai | 웹브라우저 안 (파일 업로드 가능) | 채팅 + 가벼운 분석 |
| **Claude Code** | **내 컴퓨터의 터미널** | **내 폴더에 직접 들어와서 작업** |

### 그래서 뭐가 다른데요?

- 📂 **내 파일을 직접 읽고 고침** — "이 폴더의 엑셀 파일들 정리해줘" 가능
- 🔄 **여러 작업을 연속으로 함** — 파일 읽고 → 분석하고 → 보고서까지 한 번에
- 💻 **터미널이라는 도구를 씀** — 검은 화면에 글자 쳐서 명령. 처음엔 어색해도 5분이면 적응

> 💡 **터미널이 뭐예요?** 컴퓨터에게 마우스 대신 글자로 말 거는 도구예요. 자세한 설명과 명령어는 [⌨️ 터미널 기초](/extras/terminal-basics)에, OS별 여는 법은 [🪟 Windows 풀가이드](/extras/windows-setup)에 정리해뒀어요.

## 2️⃣ 코딩 안 해도 됩니다 (정말로)

Claude Code는 이름에 "Code"가 들어가서 개발자만 쓰는 도구처럼 보이지만, 실제로는 **비개발자가 더 많이 쓰는 패턴**도 있어요.

### 비개발자도 이런 걸 시킬 수 있어요

- 📝 "이 폴더에 있는 회의록 30개 한 페이지로 요약해줘"
- 📊 "엑셀 시트들에서 매출 합계 뽑아서 새 파일로 저장해줘"
- 🌐 "이 블로그 글들을 영어로 번역해서 새 폴더에 정리해줘"
- 🗂 "다운로드 폴더 정리. 영수증은 receipts/, 이미지는 images/로"

이 모든 걸 **한국어로 부탁**하면 됩니다. 영어도, 코딩도 필요 없어요.

## 시작하기 전 체크리스트

| 필요한 것 | 안 되어 있으면? |
|---|---|
| 인터넷 연결 | (생략) |
| Anthropic 계정 (claude.ai 가입) | [공식 가입 페이지](https://claude.ai)에서 가입 |
| Pro 또는 Max 요금제 | [03 요금제와 비용](/start/pricing)에서 결정 |
| Node.js (설치 도구) | [04 설치와 첫 실행](/start/install)에서 같이 설치 |
| 노트북 (Mac/Windows 모두 OK) | — |

### Pro? Max? Cowork? 어디서 시작해야 해?

- **개인이 쓸 거다 → Pro ($20/월)** 부터 충분
- **회사에서 동료들과 → Cowork** (팀 단위)
- **하루에도 수십 번 쓸 거다 → Max ($100/월)** 추천

자세한 비교는 [03 요금제와 비용](/start/pricing)에서 해드릴게요.

## 자주 듣는 걱정

<details>
<summary><b>Q. 내 파일을 AI가 다 읽으면 정보가 새지 않아요?</b></summary>

Anthropic은 기본적으로 **사용자 데이터로 모델을 학습시키지 않습니다.** 또한 권한 시스템(Permissions)으로 "이 파일/폴더만 봐도 됨" 처럼 범위를 제한할 수 있어요. 자세한 건 [07 핵심 개념](/core/concepts)에서.
</details>

<details>
<summary><b>Q. 코딩을 진짜 1도 모르는데 따라올 수 있나요?</b></summary>

이 가이드는 **터미널 처음 열어보는 분 기준**으로 썼어요. "검은 화면에 뭘 쳐야 하지?" 부터 손잡고 갑니다.
</details>

<details>
<summary><b>Q. 무료로 써볼 수 있어요?</b></summary>

Claude Code 자체는 Anthropic 요금제에 포함된 도구라, **유료 구독이 있어야** 본격적으로 쓸 수 있어요. 다만 처음 가입하면 일정 무료 크레딧이 있을 수 있으니 [공식 사이트](https://claude.ai)에서 확인하세요.
</details>
