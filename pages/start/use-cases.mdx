## 🗂 폴더·파일 정리 (비개발자 인기 1위)

### 📥 다운로드 폴더 정리
```
이 폴더 안의 파일들을 종류별로 분류해서
- images/ : 이미지 파일
- pdfs/ : PDF 문서
- excel/ : 엑셀 파일
- 기타/ : 분류 안 되는 것들
```

## 📝 문서·텍스트 작업

### ✍️ 회의록 요약
```

주제별로 그룹핑하고, 결정 사항만 따로 모아줘.
```

### 🌐 일괄 번역
```
blog/ 폴더의 마크다운 파일들을 영어로 번역해서
blog-en/ 폴더에 같은 구조로 저장해줘.
```

### 📋 양식 채우기
```
template.docx 양식을 보고,
data.csv의 100개 행을 각각 적용해서
output/ 폴더에 100개 파일로 저장해줘.
```

## 안 되는 것 (한계점)

- ❌ 외부 결제, 이메일 자동 발송 같은 건 **MCP 도구**를 따로 연결해야 가능 ([15 MCP](/advanced/mcp))
- ❌ 클라우드에 있는 파일은 못 봄 (다운받아서 폴더에 넣어줘야 함)
- ❌ 매우 큰 파일(GB 단위)은 한 번에 다 못 읽음 — 분할 처리 필요
- ❌ 실시간으로 화면을 보는 일(예: 게임)은 못 함
