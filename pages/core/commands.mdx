# 09 · 자주 쓰는 명령어

> "/" 키 하나로 강력해지는 단축어들. 한 번 익혀두면 평생 빠릅니다.

> 📚 **이 페이지 한 눈에**
> - 외울 건 **`/help`, `/clear`, `/init`, `/exit`** 4개면 첫 한 달 OK
> - 위로 빠른 단축키: `Tab`(자동완성) · `↑`(이전 명령) · `Ctrl+C`(중단)
> - 입력 트릭: `!명령` (셸 직접 실행) · `@파일` (파일 첨부)
>
> 🔗 함께 보면 좋은 페이지: [10 세션 관리](/core/sessions) · [14 알면 편한 기능](/practice/useful-features) · [17 Skills](/advanced/skills)

## Top 10 — 이것만 외우세요

| 명령어 | 무엇을 하나요? | 언제 쓰나요? |
|---|---|---|
| `/help` | 모든 명령어 목록 | 까먹었을 때 |
| `/init` | CLAUDE.md 자동 생성 | 새 폴더 시작할 때 |
| `/clear` | 대화 컨텍스트 비우기 | 새 작업 시작 시 |
| `/exit` | Claude Code 종료 | 끝낼 때 |
| `/permissions` | 권한 모드 설정 | 위험한 도구 자동 허용 등 |
| `/model` | AI 모델 변경 | Opus → Sonnet 같은 전환 |
| `/cost` | 이번 세션 토큰 사용량 | 비용 체크 |
| `/memory` | 메모리 시스템 보기/수정 | 장기 기억 관리 |
| `/compact` | 대화 압축 (요약본으로 대체) | 컨텍스트 차오를 때 |
| `/plugins` | 플러그인 관리 | 새 기능 추가 |

## 자주 쓰는 명령어 자세히

### `/help`

지금 사용 가능한 모든 명령어를 보여줍니다. 막혔을 때 첫 번째로 칠 명령.

### `/init`

```
/init
```

현재 폴더를 분석해서 **CLAUDE.md 초안을 자동 생성**. 새 프로젝트 시작 시 먼저 이거 실행하세요.

### `/clear`

```
/clear
```

지금까지 대화한 내용을 다 지웁니다. **새로운 작업을 시작할 때** 꼭 쓰세요. 이전 작업 흔적이 남아있으면 헷갈려요.

> 💡 종료(`/exit`)와는 달라요. `/clear`는 Claude는 켜둔 채 대화만 리셋.

### `/permissions`

```
/permissions
```

지금 권한 모드 확인 + 변경. 자주 쓰는 옵션:

- **default** — 매번 묻기
- **acceptEdits** — 편집은 자동, 명령은 묻기 (편함)
- **plan** — 아무것도 안 바꾸고 계획만 (안전)

### `/model`

```
/model
```

AI 모델 선택. 일반적으로:

- **Opus** — 가장 똑똑. 복잡한 작업
- **Sonnet** — 빠르고 똑똑함. 일상 작업
- **Haiku** — 가장 빠름. 단순 작업

기본값은 보통 가장 좋은 모델. 비용 절약하려면 작업 종류에 따라 바꾸세요.

### `/cost`

```
/cost
```

이번 세션에서 얼마나 토큰을 썼는지 보여줍니다. 한도 신경 쓰는 분께.

### `/memory`

```
/memory
```

Claude의 장기 기억 시스템을 봅니다. 사용자 정보, 피드백, 프로젝트 메모 등이 저장돼있어요. [auto memory](/advanced/workspace)에서 자세히.

### `/compact`

```
/compact
```

긴 대화를 요약본으로 압축. 컨텍스트가 거의 차서 끊기기 직전 마지막 수단.

```
/compact "코드 리뷰 부분만 자세히 남기고 나머지는 줄여줘"
```

처럼 무엇을 남길지 지시도 가능.

## 명령어 외 단축키

| 단축키 | 동작 |
|---|---|
| `Ctrl + C` | 진행 중인 작업 멈춤 (1번) / 종료 (2번 연속) |
| `Ctrl + D` | 종료 |
| `Esc` | 입력 취소 / 메뉴 빠져나오기 |
| `↑` / `↓` | 이전 입력 불러오기 |
| `Tab` | 자동 완성 (파일명 등) |

## 입력 트릭

### `!` 로 셸 명령 한 번에

```
> !ls
```

라고 입력하면 `ls` 명령이 셸에서 실행되고 결과가 대화에 들어옵니다. **사용자가 직접** 명령을 친 것이라 권한 묻지 않음.

### `@` 로 파일 첨부

```
> @README.md 이 문서 요약해줘
```

특정 파일을 명시적으로 컨텍스트에 넣음.

### 여러 줄 입력

기본은 엔터 = 전송. 여러 줄을 쓰려면 `Shift + Enter` 또는 `\` 로 줄 잇기.
